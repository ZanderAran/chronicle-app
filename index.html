Setup Guide
1. Create a Google Sheet
Make a new Google Sheet and note the ID from the URL:
docs.google.com/spreadsheets/d/SHEET_ID/edit

2. Create 3 tabs
Name them exactly: checklist_items, active_timers, timer_log

3. Add headers to each tab
checklist_items Row 1:
id | title | category | location | device | tags | recurrence | days | emoji

active_timers Row 1:
id | item_id | title | category | location | device | tags | start_time

timer_log Row 1:
id | item_id | title | category | location | device | tags | start_time | end_time | duration_mins

4. Create an OAuth Client ID
• Go to console.cloud.google.com
• Enable Google Sheets API (APIs & Services → Library)
• Go to Credentials → Create Credentials → OAuth 2.0 Client ID
• Application type: Web application
• Add your hosted URL to Authorized JavaScript origins
• Copy the Client ID

5. Paste Client ID into the app file
Open index.html in a text editor and replace:
YOUR_GOOGLE_OAUTH_CLIENT_ID_HERE
with your actual Client ID, then re-upload to Netlify.
